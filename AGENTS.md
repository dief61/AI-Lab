# Projekt-Anweisungen für OpenCode

## Tech Stack
- **Sprache:** Python (Paketverwaltung mit `uv`)
- **Datenbank:** PostgreSQL (erstelle einen Docker-Container `postgres-dev`)
- **IDE:** VS Code unter Windows via WSL2 Remote Integration

## Regelmuster & Standards
- Verwende `uv add <package>` zum Hinzufügen neuer Pakete.
- Nutze `asyncpg` oder `psycopg2` für DB-Abfragen.
